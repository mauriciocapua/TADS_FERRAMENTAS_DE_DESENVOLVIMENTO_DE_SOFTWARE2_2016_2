Instalar no Ubuntu:
1) O python já vem instalado em distribuições linux
2) Instalar PIP: 
- sudo apt-get -y install python-pip
3) Instalar Flask:
- sudo pip install Flask
4) Instalar suporte Python com PostgreSQL:
- sudo apt-get install python-psycopg2

Instalar no Windows:
1) Instalar Python
- https://www.python.org/downloads/
- Habilitar classpath (o único "x" vermelho que não está marcado na instalação)
- Adicionar nas variáveis de ambiente
2) Instalar PIP: 
- Baixar o arquivo get-pip.py
- Site: https://pip.pypa.io/en/stable/installing/
- E executá-lo: python get-pip.py
3) Instalar Flask:
- pip install Flask
Link de apoio: http://matthewhorne.me/how-to-install-python-and-pip-on-wi…/
4) Instalar suporte Python com PostgreSQL:
- http://www.stickpeople.com/projects/python/win-psycopg/
